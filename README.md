# mio2's setup environments for Linux Branch

## for ubuntu
```bash
chmod +x *
./ubuntu.sh
```

## Contains
- zsh
    - oh-my-zsh
    - zsh-completions
    - zsh-autosuggestions
- byobu
- linuxbrew

- wsl2 display support (WSL2 only)

## addon
1. mkdir ~/conf/addon
2. set config fils on ~/conf/addon/(example).sh
