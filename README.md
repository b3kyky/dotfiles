# mio2's setup environments for Linux Branch

- zsh
- curl
- wget

## Contains
- zsh
    - oh-my-zsh
    - zsh-completions
    - zsh-autosuggestions
- byobu
- linuxbrew

- wsl2 display support (WSL2 only)
