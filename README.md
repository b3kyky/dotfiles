# nvim cfg

## LSP

### bash : bashls
 - common : `npm install --location=global bash-language-server`

### C/C++ : clangd
 - windows : `scoop install llvm`
 - POSIX : `brew install llvm`

### Python : Pyright
 - common : `npm install --location=global pyright`

### SystemVerilog : svls
- common : `cargo install svls`
